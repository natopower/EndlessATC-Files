# EndlessATC-CYEG
A file for Edmonton International Airport (CYEG) to be used in EndlessATC.

# Includes the following fully-modeled arrivals:
- ELLKS3
- IGSOX3
- OILRS2
- RESAX1
- TETAG2

And all RNAVZ and RNAVY approaches as of January 2026.

Use it by dragging the .txt file into your EndlessATC's "Locations" folder (likely in something likeC:\Program Files (x86)\Steam\steamapps\common\Endless ATC\locations).
